# RHEL & Fedora Configuration  
  
## Automated Install  
  
```shell
sudo bash -c "$(curl -sSL https://github.com/pkmgr/centos/raw/main/scripts/default.sh)"
```

## Automated Update  

```shell
sudo bash -c "$(curl -sSL https://github.com/pkmgr/centos/raw/main/root/.local/bin/run-os-update)"
```
